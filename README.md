# Airbnb Clone

A full-stack Airbnb-inspired hotel booking platform built with Java Spring Boot and React.js.

## Tech Stack

**Frontend:** React.js, Tailwind CSS  
**Backend:** Java, Spring Boot, Spring Security, Spring Data JPA  
**Database:** PostgreSQL  
**Auth:** JWT-based authentication  

## Project Structure
airbnb-clone/
├── client/   # React frontend
└── server/   # Spring Boot backend
## Live Demo

- Frontend: [airbnb-frontend-rose.vercel.app](https://airbnb-frontend-rose.vercel.app)
- Backend: Deployed on Render

## Features

- User authentication (register/login)
- Browse and search hotel listings
- Booking management
- Role-based access (guest/host)
